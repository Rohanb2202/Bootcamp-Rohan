Exercise 1

  1. Level 1:
    • Create a web page (solution.html) that includes:
    • A header with the title: "My First Webpage".
    • A footer with contact info: "Contact: example@email.com".
    • A main content area with:
      • An ordered list of three favorite movies.
      • An unordered list of three favorite books.
      • One image of your choice (use an accessible URL).
    • Use semantic elements like <section>, <article>, and <aside>.
  2. Level 2:
    • Add a golden layout structure: header, two sidebars, and a main content area.
    • Use <nav> for the sidebars to create a simple navigation menu with three links.
    • Include a table listing three items with columns for name, price, and description.
  3. Level 3:
    • Add dummy text to each section to make it look more complete.
    • Ensure proper nesting and alignment of all elements.
    • Make the content accessible by using alt text for images and clear labels for table headers.

My Approach

I started by thoroughly reading and understanding the requirements for each level of the exercise. For Level 1, I focused on using semantic HTML elements like <header>, <footer>, and <section> to structure the webpage. I ensured that the content was simple and readable, with ordered and unordered lists styled appropriately. I added an image using a publicly accessible URL and provided alt text for accessibility.

For Level 2, I worked on creating a golden layout structure by dividing the page into a header, two sidebars, and a main content area. I used the <nav> element to create a simple navigation menu in the sidebars and added a table with well-defined columns for items, their prices, and descriptions.

In Level 3, I enriched the content by adding meaningful dummy text to make the page look complete and professional. I carefully ensured proper nesting and alignment of all elements and added alt attributes to images and clear labels to table headers to make the page more accessible. This step ensured compliance with web accessibility standards.

Throughout the process, I tested the webpage regularly in a browser to verify its layout and functionality. I refined the code where needed to maintain clean and readable HTML.
