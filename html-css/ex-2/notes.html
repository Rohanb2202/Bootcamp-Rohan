Exercise 2

Level 1: Defining CSS Variables

In this step, five CSS variables were defined in a variables.css file. The variables include:

  • --primary-color: Used for the main color theme of the page.
  • --secondary-color: Complementary color for accents.
  • --header-font-size: Specifies the font size for headers.
  • --body-font-size: Sets the font size for body text.
  • --margin-spacing: Defines consistent spacing for margins.

The variables were linked to the main CSS file and applied to style headers, body text, and spacing in the HTML structure from Exercise 1.

Level 2: Advanced CSS Variable Usage

To ensure flexibility, default values were set for the variables using the var() function. For example:

color: var(--primary-color, black);

Alternate rows in the table were styled using the :nth-child selector:

tbody tr:nth-child(even) {
    background-color: var(--secondary-color, #f0f0f0);
}

Level 3: Dynamic Theme Switching

Variables were defined dynamically, such as:

--secondary-color: var(--primary-color);

A theme switcher was implemented using JavaScript. The theme toggles between light and dark modes by modifying the data-theme attribute on the <body> tag. Example CSS:

[data-theme="dark"] {
    --primary-color: #222;
    --secondary-color: #555;
    --header-font-size: 1.5rem;
    --body-font-size: 1rem;
}

Challenges Faced

  • Ensuring cross-browser compatibility for CSS variables.
  • Creating a seamless transition between light and dark modes.
  • Maintaining readability with alternate row styling.
