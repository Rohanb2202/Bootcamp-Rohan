Exercise 3: Starting with Vite

Objective

Learn how to use Vite to scaffold, configure, and build projects with modern tools.

Steps

  1. Starter Template (JavaScript):
    • Initialize a Vite project using the JavaScript template:
      • npm create vite@latest my-js-app -- --template vanilla
    • Task: Rebuild the student management dashboard from Exercise 1:
      • Load student data from a JSON file and display it in a table.
  2. Starter Template (TypeScript):
    • Initialize a Vite project using the TypeScript template:
      • npm create vite@latest my-ts-app -- --template vanilla-ts
    • Task: Enhance the student management dashboard:
      • Fetch data from a CSV file, parse it using PapaParse, and display it using AG Grid.
      • Modularize the code for data loading, parsing, and rendering.
  3. Vite with Extended Options:
    • Define a CSS file with classes like m-1, m-2, until m-8, where each m-n is n*2px.
    • Recreate the same classes in SCSS using loops.
    • Utilize Viteâ€™s SCSS support to configure and integrate the SCSS file.

My Approach

  1. Starter Template (JavaScript):
    • Initialized the project using npm create vite@latest my-js-app -- --template vanilla.
    • Rebuilt the student management dashboard by:
      • Loading student data from a JSON file using fetch.
      • Rendering the data in a dynamically generated HTML table.
  2. Starter Template (TypeScript):
    • Initialized the project using npm create vite@latest my-ts-app -- --template vanilla-ts.
    • Enhanced the dashboard by:
      • Fetching and parsing CSV data using PapaParse.
      • Displaying the data in an AG Grid table with features like column sorting and filtering.
      • Separating the logic into modules for data loading, parsing, and rendering.
  3. Vite with Extended Options:
    • Created a CSS file and used SCSS to define the same classes dynamically with loops with the help of documentation and LLM
    • Integrated SCSS into the Vite project by:
      • Installing SCSS support: npm install sass.
      • Creating and linking the SCSS file in the project.
