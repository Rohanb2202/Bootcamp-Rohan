Exercise 4: Basics of Images

Level 1

  • Add three images to a web page.
  • Apply CSS styles for width, height, and border-radius.
  • Use a placeholder image service (e.g., Lorem Picsum).

Level 2

  • Edit an image to adjust its size and compression.
  • Add a full-bleed image as a page header.
  • Use CSS to add overlays (e.g., gradient or color tint).

Level 3

  • Implement responsive images using the <picture> element.
  • Create an image with high-res and low-res versions for different devices.
  • Optimize image loading using lazy loading techniques.

My Approach

Level 1

For the first level, I added three images to the page using Lorem Picsum as the placeholder image source. I applied CSS properties like width, height, and border-radius to style the images, ensuring consistency and a polished appearance.

Level 2

In the second level, I edited images to optimize their size and compression for faster loading times. I added a full-bleed image at the top of the page as a header and used CSS to apply gradient overlays, enhancing the visual appeal and readability of overlaid text.

Level 3

At the advanced level, I implemented responsive images using the <picture> element to serve different image sizes for various devices. I created high-resolution and low-resolution versions of images to optimize bandwidth usage. Additionally, I incorporated lazy loading to improve page performance by deferring the loading of offscreen images.
