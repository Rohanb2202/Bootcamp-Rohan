Exercise 1: Basics of Fonts

Level 1

  • Add a snippet of JavaScript code (10 lines).
  • Apply basic font styles using system fonts (e.g., Arial, Times New Roman, Courier).
  • Change the weights and apply different weight to different classes. Update the page to inform which part has which weight of font.

Level 2

  • Use Google Fonts to apply three fonts (sans-serif for text, serif for headings, monospace for code).
  • Check how variable fonts work and illustrate them on your page.
  • Show what you can do with text: Drop cap, shadow, and large headings where an image shows only through the text.

Level 3

  • Install a custom font locally and use it in the page (e.g., Geist font from Vercel).
  • Combine multiple font families to create a typographic hierarchy (e.g., headers vs. body text).
  • Adjust font height, spacing, etc., and illustrate on the same page with additional paragraphs.

My Approach

Level 1

To start, I used system fonts like Arial, Times New Roman, and Courier New. I applied font weights using CSS classes and ensured that the updated weights were reflected in the content. Each font type and weight was documented to make its application clear.

Level 2

Next, I integrated Google Fonts to apply three distinct fonts. Sans-serif fonts were used for text, serif for headings, and monospace for code. I explored variable fonts and demonstrated their utility by dynamically changing font weights and styles. Additionally, I implemented text effects like drop caps, shadows, and large headings with background images visible only through the text.

Level 3

I installed a custom font (Geist) locally and used it for a unique style. I also combined multiple font families to establish a clear typographic hierarchy. Font height and spacing were adjusted to enhance readability and aesthetic balance. New paragraphs were added to demonstrate these changes effectively.
