Exercise 2: Basics of Colors

Level 1

  • Add a card to the page. Apply a monochromatic theme to the entire page using a base color. Ensure the card (background, border, and shadow) stands out using alpha channels for subtle differentiation.
  • Create a second version of the page using a complementary color scheme. Add clear contrast between elements.
  • Extend to a three-color scheme by incorporating an accent color. Style headers, links, and buttons with the accent color.
  • Save each version as a separate file (e.g., solution-2-1.html, solution-2-2.html).

Level 2

  • Create dark and light themes for the page using CSS variables:
    • Define --primary, --secondary, and --background colors.
    • Create two separate stylesheets, one for dark and one for light mode.
  • Style an image with overlaid text:
    • Adjust text and overlay colors for contrast in both themes.
  • Save two versions of the page (dark and light).
  • Build a color swatch page:
    • Use HSL values to generate variations in saturation and lightness.
    • Create a grid of color swatches labeled with their HSL values.

Level 3 (Advanced)

  • Design a gradient theme using only CSS:
    • Apply linear gradients to headers and radial gradients to buttons.
    • Save a version showcasing how gradient angles and stops affect the design.
  • Create a responsive color palette:
    • Define tints and shades for the base colors using lighten() and darken() in CSS.
    • Apply the palette dynamically using media queries for different breakpoints.
  • Implement a contrast demonstration page:
    • Create sections showing how different color combinations meet accessibility standards.
    • Use borders and shadows to enhance visual hierarchy.

My Approach

Level 1

I started by applying a monochromatic theme using a base color to ensure visual consistency. To make the card stand out, I used alpha channels to subtly differentiate its background, border, and shadow. For the complementary color scheme, I selected contrasting hues to improve readability and user engagement. Finally, I incorporated an accent color to highlight headers, links, and buttons, ensuring a cohesive design across the page.

Level 2

Using CSS variables, I defined --primary, --secondary, and --background colors to create dark and light themes. Separate stylesheets were designed for each mode, allowing users to toggle between them. For the image with overlaid text, I ensured adequate contrast in both themes by carefully selecting overlay and text colors. The color swatch page was built using HSL values, showcasing variations in saturation and lightness in a well-structured grid.

Level 3

I created a gradient theme, applying linear gradients to headers and radial gradients to buttons. This showcased how gradient angles and stops can enhance design aesthetics. A responsive color palette was developed using CSS functions like lighten() and darken() to define tints and shades, dynamically applied using media queries. Finally, I implemented a contrast demonstration page to highlight accessibility-compliant color combinations, using borders and shadows to establish a clear visual hierarchy.
