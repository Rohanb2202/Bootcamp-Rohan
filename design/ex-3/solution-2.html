So What is This About?

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the CSS file into this page. The HTML remains the same, the only thing that has changed is the external CSS file. Yes, really.

Participation Requirements

We only ask that you submit original designs. Modified versions of existing designs will not be accepted. This is about creative CSS design; submit something unique. And by submitting, you agree to release your CSS work under the MIT License. As in, you retain ownership, but others can freely use your work to learn from and experiment with.

Benefits of CSS Design

Using CSS, you can create stunning visual designs without altering the HTML content. This separation of content and style is crucial for responsive design and accessibility.
