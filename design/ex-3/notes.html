Exercise 3: Basics of Icons

Level 1

  • Add three icons to a page using an icon font library (e.g., Font Awesome).
  • Style the icons using CSS (e.g., size, color).

Level 2

  • Replace one icon with an SVG file.
  • Create a grid of six icons (three from a font library and three as inline SVG).
  • Add hover effects to icons (e.g., color change, scaling).

Level 3

  • Design a navigation bar with icons as the main interactive elements.
    • Include home, about, and contact icons.
    • Align the icons with labels using CSS.
  • Create buttons for download and upload, embedding SVG icons.
  • Apply consistent spacing and alignment to all icons on the page.

My Approach

Level 1

I began by integrating an icon font library, such as Font Awesome, to add three distinct icons to the page. Using CSS, I styled the icons to ensure appropriate sizing and color coordination, making them visually appealing and consistent with the page theme.

Level 2

To enhance the design, I replaced one of the font icons with an inline SVG. I created a grid showcasing six icons, evenly distributed with three from the font library and three as SVGs. For interactivity, I added hover effects such as color changes and scaling to make the icons more engaging for users.

Level 3

I developed a navigation bar where icons served as the primary interactive elements, aligning them with labels for home, about, and contact sections. I also created buttons for upload and download functionalities, embedding SVG icons within them. Consistent spacing and alignment were applied across all icons, ensuring a professional and polished layout.
